#  EMACS GOOGLE WORKSPACE CHEAT SHEET

##  GLOBAL SHORTCUTS (Use Anywhere)
Press these keys from any file or buffer inside Doom Emacs to launch your tools.

  SPC g t    ->  Open Google Tasks Dashboard
  SPC g q    ->  Quick Add Task (without leaving your current file)
  SPC g c    ->  Open Google Calendar View

---

##  GOOGLE TASKS COMMANDS
Use these single-letter keys when inside the *Google Tasks Dashboard* buffer.

  a          ->  Add Task (Prompts for title at bottom, uploads instantly)
  c / RET    ->  Complete Task (Marks the task under cursor done)
  g          ->  Refresh Sync (Force pull latest data from your phone)

---

##  GOOGLE CALENDAR CONTROLS
Use these keys when inside your interactive *Calendar* buffer.

   Navigation
  h / l      ->  Move Day Left / Right
  j / k      ->  Move Week Down / Up
  v          ->  Change View (Day / Week / Month / Year)
  g / t      ->  Jump to Today

   Event Management
  a          ->  Add New Event
  e / c      ->  Edit Event Details
  d          ->  Delete Event
  q          ->  Close Calendar
